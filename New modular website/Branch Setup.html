Branch Setup


This guide provides step-by-step instructions on how to set up a branch on Uncia Studio. It covers adding branch details, selecting the branch type, entering basic details, inputting the address, and providing contact information. Through branch setup, users can create bank branches as per requirement for disbursement and collecting payments. Following this guide will ensure a smooth and accurate branch setup.


Raghuram | 9 steps | 2 minutes


Click on "New Branch" to add branch details.

2 Select the type of branch from the LOV and enter the branch name.

Click Continue. More details about the branch will be required to key in.

Enter the basic details - Alternate Region code (code used internally within the organization) and GSTIN.

Select the appropriate value from the list of values for other mandatory details:

• "High level" code: Enter the office code to which this office is attached.For example: if the office is Regional Office, then it is attached to Zonal Office.
• Profit Center: Enter the unique identifier for the branch's financial unit.
• Choose the Start date: Transaction may be carried out in this branch, only on or after the start date.
• Branch Segment: Select the category based on branch's geographical location: urban, rural or semi-urban.
• Calendar: Defines working days of the organization (Pre-requisite: defined in calendar setup)
• Status: Choose active if branch is valid. Select in-active if branch is no longer in use.

Input the address of the branch defining the location.

Enter postal code and other details auto-populates.

7 Click save and continue to move to contact details tab.

Enter name, email ID, mobile number and designation of the concerned personnel to reach out within the branch.

Click Submit.
