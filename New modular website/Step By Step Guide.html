Step-by-Step Guide: Cancelling and Viewing Deviation Level for Financed Instruments


This module facilitates the cancellation of already financed instruments. A disbursement can be cancelled using Programme ID and/or Batch ID for the disbursement.


Raghuram | 8 steps | 35 seconds


To cancel a disbursement, there's a tab "Disbursement Cancellation" in the Bill Discounting screen.

2 To initiate cancellation for a new transaction, click "Disbursement Cancellation".

Enter the batch ID or the programme name for the disbursement that is required to be cancelled in the "Search" field

Click on "Select" to select the batch ID that is to be cancelled.

The transaction details for that particular batch ID will appear in the grid. Deviation level can be checked by clicking on "View Deviation Level".

Click "View Deviation Level". The deviation details, if any, for this batch ID will appear in this pop-up. After that, close the tab.

7 Click on "SUBMIT".

Click "Ok".
