Menu Access

This guide provides step-by-step instructions on how to give and edit menu access in Uncia Studio. It explains how to navigate to the menu access tab, view and edit existing settings, and select menus to give access to a specific role. It highlights the importance of creating roles beforehand and emphasizes the simplicity of the process by just clicking checkboxes. The guide concludes by instructing users to submit their selections once they are done


Raghuram | 3 steps | 19 seconds


The list of Roles is displayed on the Menu Access screen.
Click "Edit" button on the respective Role that you want to give the Menu Access to.
Click "View" button to view the existing settings of the Role with respect to Menu Access.
Prerequisite: Creation of Role in Routine Setup --> Role

Select the Menus to give the access to the user.
All selected Menu items will be visible to the respective Roles when logged in to the UNCIA Chain Platform. Focused menus simplify navigation, safeguard sensitive data and created seamless user experience.

Click "Submit".
