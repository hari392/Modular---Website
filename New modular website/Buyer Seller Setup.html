Buyer/Seller Setup


This guide provides step-by-step instructions on how to set up a buyer/seller in a program. It covers selecting program details, entering buyer/seller information manually or through file upload, configuring instrument parameters, setting interest rates and penalties, adding co-borrowers and security details, and including additional parameters like charges and corporate codes. Following this guide will ensure a smooth and accurate setup process for buyers and sellers in a program.
