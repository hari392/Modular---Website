Direct Excess


Direct excess is the surplus amount received from the customer. This guide provides step-by-step instructions on how to create a direct excess record. It includes adding and submitting the excess details. Following this guide will ensure a smooth creation of excess details record.


Raghuram | 11 steps | 4 minutes


Click on "New Direct Excess" button for creating a direct excess record.

Select the appropriate mode of payment from the list.

Select the appropriate value from the dropdown.

Select the Bank Name..

Select the applicable bank name from the list.

Select the Customer Name.

Select the customer from the list.

Click on continue to go to the Direct Excess Details.

The value date will be determined as the date on which the surplus amount is
received from the customer.
Select the value date.

Enter the surplus amount received from the customer.

Click submit for recording the excess details.
